# CD-11 June 2026 Primary — Results & Turnout

Interactive results map for California's 11th Congressional District primary (June 2, 2026). Open `index.html` in a browser, or enable GitHub Pages on this folder to publish it.

Vote totals are official SF Department of Elections results. Turnout-by-group and 'how groups voted' figures are modeled estimates, labeled as such. Aggregate data only — no individual voter information.
